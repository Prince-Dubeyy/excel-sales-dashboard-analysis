# 📊 Vrinda Store Sales Dashboard (Excel Project)

## 🧾 Overview
This project analyzes **2022 e-commerce sales data** using Microsoft Excel and presents insights through an interactive dashboard.

The dashboard helps in understanding **sales performance, customer behavior, and channel contribution**.

---

## 🔍 Key Insights
- Women contributed approximately **64% of total sales**
- **Amazon** was the top-performing sales channel
- Identified **monthly sales and order trends**
- High delivery success rate (~90%+)
- Top contributing states include **Maharashtra, Karnataka, and Uttar Pradesh**

---

## 🛠 Tools & Techniques
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers (Interactive Filters)  
- Data Cleaning & Transformation  
- Data Visualization

- Dynamic filtering by **Month, Channel, and Category**
